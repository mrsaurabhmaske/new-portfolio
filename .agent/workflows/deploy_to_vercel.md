# Deploy to Vercel

This workflow deploys the current project to Vercel.

1.  **Login to Vercel** (if not already logged in)
    ```bash
    npx vercel login
    ```

2.  **Deploy to Production**
    ```bash
    npx vercel --prod
    ```

    Follow the prompts to link the project if this is the first time.
